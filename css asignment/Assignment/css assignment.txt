1) What are the benefits of using CSS?
Ans. - HTML just provides structure of the web page, while CSS                     provides look and feel, with CSS we can make eye catching websites.
-   By using CSS we can also create responsive designs that adapt to various devices. CSS also allows animations and transitions for enhancing the user experience.

2) What are the disadvantages of CSS?
Ans. - Developers need to write browser specific CSS code, because the visual representation of web page may be different for different browsers.
-	Complex for beginners as it has lots of properties so remembering all the properties sometimes may seems difficult for a developer.

-	CSS does not provides any security features. So developers must be cautious while implementing it.

-	It’s essential to minimize and optimize CSS code otherwise poorly written or too much lengthy CSS codes can affect the website performance by slowing it down.

3) What is the difference between CSS2 and CSS3?
Ans.  -  CSS2 has limited selectors options, while CSS3 introduced some new useful selectors like attribute, nth – child and adjacent sibling selectors that allows you to select specific elements more easily.
-	CSS2 introduced a new box model which included padding, borders, and margins. CSS3 introduced the flexible box layout module, which allows for more flexible layouts.

-	CSS3 introduced new animation and transition properties, such as transition, animation, and key frames, which allow for more dynamic and engaging web designs.

4) Name a few CSS style components
Ans. - Here are some common style components of CSS :
•	Selector: Class name, id name or element name that is target.
•	Attribute: Name of the attribute you want to style for example border, colour, background, position etc.
•	Value of Property: Value that will be assigned to attribute.

ex : p {
	  color : blue;
	 text-align : left;

(Here p is selector, color is attribute and value is blue.)


5) What do you understand by CSS opacity?

Ans. - The opacity CSS property sets the opacity of an element. Opacity is the degree to which content behind an element is hidden.

ex : background-color : rgba(red,green,blue,0.5);

(Here red, green and blue are the values of the color and 0.5 is the alpha value which is also mentioned as opacity of that color.)

6) How can the background color of an element be changed?

Ans. - We can change the background color of an element by using it’s name or it’s class name or it’s id name.

ex:  p {
             background-color : blue;
            }
          


7) How can image repetition of the background be controlled?

Ans. – In CSS there is a property called background-repeat property by using this we can control the repetition of the image in the background.

ex :  div{
	       background-repeat : no-repeat;
               }


8) What is the use of the background-position property?

Ans. - CSS background position property sets the initial position for each background image.

-	It has total  values : left top, left center, left bottom
      : right top, right center, right bottom
      : center top, center center,
        center bottom

ex : body{
		background position : center top;
                  }
9) Which property controls the image scroll in the background?

Ans. - The background-attachment property of CSS sets whether a background image scrolls with the page or will be fixed in the background.

-	It has several values like scroll, fixed, local, initial and inherit.

ex :  body  {
                     background-image: url("img.jpg");
                     background-repeat: no-repeat;
                     background-attachment: fixed;
                     }


10) Why should background and color be used as separate   properties?

Ans. - Background is a shorthand property that allows you to set all the background properties including the background-color in one line.

-	While background-color property will sets only the color of the background of an element.


ex : div{
              	background : #f9f9f9 url(image1.jpg);
              }

(Here by using only background property we can write multiple background properties separated by space.)


ex: div{
		background-color : #f9f9f9;
              }

(Here by using background-color property we can only set background color of the element.)


11) How to center block elements using CSS1?

Ans. – For centring the block elements in CSS1 we have to use margin property. We don’t have to set the values of margin manually, Instead we can use auto property of margin to center the block elements.

ex : div{
		height : 200px;
		width : 200px;
		background-color : blue;
		margin : auto;
              }



12) How to maintain the CSS specifications?

Ans. – CSS specifications can be maintained by the following :

1.	Positioning
2.	Box Model
3.	Typographic
4.	Visual



13) What are the ways to integrate CSS as a web page?

Ans. – There are 3 ways to integrate CSS as a web page :

1.	Inline : To style a single element in HTML, we can directly write style attribute within the tag and give properties to it.

ex : <p style=”color : blue;”> This is paragraph. </p>


2.	Internal : By adding CSS to the head section of our HTML document, we can embed an internal stylesheet.

ex : <head>
        <style>
		p {color : blue;}
        </style>
        </head>

        <body>
		<p> This is paragraph. </p>
        </body>


3.	External : First creating the separate file with .css extension then add it’s path in link tag in HTML.

ex : p{color : blue;}

Then, add the link tag in head section of the HTML page.

<head>
<link href = “style.css” rel = “stylesheet”>
</head>

<body>
<p> This is paragraph. </p>
</body>



14) What is embedded style sheets?

Ans. - It allows you to define styles for a particular HTML document as a whole in one place. This is done by embedding the <style></style> tags containing the CSS properties in the head of your document.

ex: 

<head>
<style>
		h1{
			color : green;
    }

p{
	background-color : blue;
}
         </style>
</head>



15) What are the external style sheets?

Ans. - An external style sheet is a separate CSS file that can be accessed by creating a link within the head section of the webpage.


ex : <head>
        	<link href = “style.css” rel = “stylesheet”>
        </head>


16) What are the advantages and disadvantages of using external style sheets?

Advantages : 
-	One change to the style sheet will change all linked pages
-	You can create classes of styles that can then be used on many different HTML elements
-	Consistent look and feel across multiple web pages
-	Improved load times because the css file is downloaded once and applied to each relevant page as needed
Disadvantages :
-	While implementing style sheets, we need to test Web pages with multiple browsers in order to check compatibility issues
-	An extra download is essential to import style information for each file
-	The execution of the file may be deferred till the external style sheet is loaded


17) What is the meaning of the CSS selector?
Ans. - CSS selectors are used to define the elements you want to style with CSS. There are many different types of CSS selectors, each with their own unique syntax. These tell the browser which elements to apply CSS property values to.
-	Some commonly used CSS selectors are : ID selector, Class selector, Universal selector, Group selector, Pseudo selectors etc.

18) What are the media types allowed by CSS?
Ans. - Given below is a list of various media types :
1.	All – For all devices
2.	Aural – For speech synthesizers
3.	Braille - For braille tactile feedback devices
4.	Embossed - For paged braille printers
5.	Handheld - For handheld devices
6.	Print - For documents viewed on screen in print preview mode
7.	Projection - For projectors
8.	Screen - For computer screens
9.	Tty and Tv - For portable & television type devices

19) What is the rule set?
Ans. - A CSS rule set contains one or more selectors and one or more declarations.
ex :  h1 {					
		color : blue;
		text-align : center;
    }
(Here h1 is selector, color and text-align are properties, blue and center are values of that properties.)
